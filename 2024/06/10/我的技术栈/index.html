小四的技术栈

2024/6/10

简介

我的技术栈包括了从开发工具到数据库，从后端框架到容器化管理的全方位技术支持。我能够熟练使用 IDEA 和 VS Code 进行开发，利用 Maven 管理项目，结合 Spring 和 Spring Boot 开发高效、可扩展的应用程序。通过 MyBatis-Plus 简化数据库操作，并在 MySQL 数据库中实现高效的数据持久化管理。同时，熟练使用 Tomcat 作为Web容器，结合 Docker 进行应用容器化，确保开发与部署的高效性与一致性。这些技术栈不仅让我具备了扎实的开发能力，也让我在开发中能够更加高效地处理复杂的业务需求和技术挑战。

技术栈介绍

  • 1. 开发工具

    • IDEA：作为主力开发工具，IDEA具备强大的代码自动补全、调试和测试功能，极大提高了开发效率。我熟悉使用IDEA进行Java项目的开发、调试和版本控制。
    • VS Code：作为轻量级的代码编辑器，VS Code支持多种语言和插件扩展，适用于快速开发和调试，尤其在处理Web前端和轻量级后端服务时表现出色。

    2. 项目管理与构建

    • Maven：我熟练使用Maven进行项目管理，尤其擅长使用Maven进行依赖管理、构建、打包和发布。Maven简化了项目的构建过程，确保了项目的一致性和可维护性，适用于多模块项目的管理。

    3. Java基础编程

    • Java语言基础：我具备扎实的Java基础，熟悉面向对象编程（OOP）、集合框架、多线程、异常处理、文件IO等核心知识，并能够在开发中灵活应用这些技术。

    4. Spring框架

    • Spring框架：我深入理解Spring框架，能够使用Spring的依赖注入（DI）和面向切面编程（AOP）等功能来设计高效、解耦的企业级应用。通过Spring的管理和事务控制，能够提升代码的可维护性和扩展性。

    5. Spring Boot

    • Spring Boot：熟练使用Spring Boot构建快速、稳定的Spring应用。Spring Boot通过自动配置和开箱即用的功能，简化了项目配置和部署流程，使我能够专注于业务逻辑的开发，快速交付高质量的应用。

    6. MyBatis-Plus

    • MyBatis-Plus：我熟悉使用MyBatis-Plus简化数据持久化操作，能够快速完成基本的CRUD功能，并通过其强大的条件构造器和分页插件简化查询逻辑，提高了开发效率。

    7. 数据库与持久化

    • MySQL：我熟练使用MySQL进行数据建模和管理，能够编写高效的SQL语句，优化数据库性能，设计数据表关系以及执行数据库迁移等操作。同时，熟悉MySQL的索引、事务处理、视图等高级功能，确保数据库的高可用性和高性能。

    8. Web应用与容器

    • Tomcat：我熟悉Tomcat作为Java Web服务器的配置与使用，能够在Tomcat上部署和调试Spring Boot应用。熟悉Tomcat的日志管理、连接池配置和性能优化，确保Web应用的稳定运行。

    9. 容器化与自动化部署

    • Docker：我掌握Docker容器技术，能够使用Docker进行应用的容器化部署，确保开发、测试和生产环境的一致性。通过Docker，能够轻松打包、分发和部署应用程序，提升开发效率，并支持微服务架构的容器化管理。

    10. 版本控制

    • Git：我熟练使用Git进行版本控制，能够高效管理代码版本、进行代码分支管理、解决冲突，并通过Git进行团队协作开发。Git帮助我在多人的开发环境中保证代码的稳定性和可追溯性。
